= UML Feature To Code

== No tags, no statements

=== Given The mbt-transformer plugin, src/test/resources/cucumber/Process.feature file is created as follows

----
Feature: Process
----

=== And The mbt-transformer plugin, uml-to-cucumber goal is executed

| Name
| Process
